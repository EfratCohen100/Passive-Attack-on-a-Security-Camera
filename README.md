## Working Environment:
The code was written in python 3.6.8 in Linux operating system.

## Requirements:
* Wifi Adapter Card supports the Linux operating system.
* Security Camera
* Python Libraries - aircrcak-ng, scapy


## Run the code:

* clone this project
* `sudo su`
* `python3 project.py`
